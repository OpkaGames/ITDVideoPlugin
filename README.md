# ITDVideoPlugin
Плагин добавляющий возможность отправлять и смотреть видео в ИТД.

[!CAUTION]
ТОЛЬКО ДЛЯ БРАУЗЕРОВ НА ОСНОВЕ CHROMIUM (Chrome, Edge, и т.д)

## Туториал по установке плагина
![Фотография, указывающая где включать режим разработчика](/images/1.png)
# Письменный туториал
1) Перейдите на chrome://extensions (или edge://extensions и т.д)
   Если не подходят, попробуйте найти меню с расширениями и перейти туда
2) Включите режим разработчика
3) Перенесите архив с расширением из папки загрузок в данное окно
4) Перезапустите браузер
5) Пользуйтесь!
